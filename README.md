# C# Console Apllication - User Interaction APP

Overview
This is a simple C# console application that demonstrates basic input and output operations. The program greets the user, asks for their name, and responds with a personalized message.

This project was created as part of an in-class assignment to practice core C# programming concepts.

Features
 Displays text using `Console.WriteLine()`
 Accepts user input using `Console.ReadLine()`
 Uses string interpolation for dynamic output
 Simple and clean user interaction

During this process, I learned How to create and run a .NET console application, how to use Console.WriteLine() and Console.ReadLine(). I Learned the how to implement basic variable declaratiom and user input handling, 
and I learned how to implement string interpolation in C#.

How to Run the Project
git clone https://github.com/YOUR-USERNAME/csharp-console-app.git

Navigate to the project folder
cd csharp-console-app

Run the application
dotnet run
